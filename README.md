# vue-todo 앱 만들기
인프런 강의를 보고 Vue.js 학습하는 Repository입니다.

###  필요한 패키지 설치
```
npm install
```

### 개발을 위한 Compiles & hot-reloads
```
npm run serve
```

### 배포를 위한 compile & 압축(minify)
```
npm run build
```

### lint fix 명령어 
```
npm run lint
```
